This package uses synonyms defined by WordNet to build a Lucene index storing them, which in turn can be used for query expansion. You normally run {@link org.apache.lucene.wordnet.Syns2Index} once to build the query index/"database", and then call {@link org.apache.lucene.wordnet.SynExpand#expand SynExpand.expand(...)} to expand a query.

Instructions

  1. Download the WordNet prolog database , gunzip, untar etc.
  2. Invoke Syn2Index as appropriate to build a synonym index. It'll take 2 arguments, the path to wn_s.pl from that WordNet downlaod, and the index name.
  3. Update your UI so that as appropriate you call SynExpand.expand(...) to expand user queries with synonyms.
